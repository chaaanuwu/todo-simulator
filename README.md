# Todo List Simulator

A simple **Todo List application** built with **Node.js, Express, and EJS**, featuring:
- Add, update, and delete tasks.
- Tasks persisted in a JSON file.
- Responsive UI using **Bootstrap 5** and **FontAwesome** icons.

## Features

- **Add Task**: Add a new task.
- **Update Task**: Edit an existing task.
- **Delete Task**: Remove tasks with a single click.
